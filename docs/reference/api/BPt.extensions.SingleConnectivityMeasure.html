BPt.extensions.SingleConnectivityMeasure¶

class BPt.extensions.SingleConnectivityMeasure(cov_estimator=LedoitWolf(store_precision=False), kind='covariance', vectorize=False, discard_diagonal=False)[source]¶
See nilearn.connectome.ConnectivityMeasure. This class is just a wrapper to let this object work when passed a single connectivity matrix.
This class requires extra dependency nilearn to be installed.
